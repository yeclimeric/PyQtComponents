D1: 设计风格定位

决策：保持系统原生优先，但增强视觉一致性

核心原则

  • • 系统原生风格优先
  • • 尊重操作系统主题
  • • 功能导向设计
  • • 跨平台一致性

目标用户

  • • 科研人员
  • • 电生理学家
  • • 数据分析师

技术实现

  • • Qt QPalette
  • • Fusion 风格
  • • 系统字体栈

D10: 反馈和状态指示

决策：采用分层反馈策略

反馈类型

Toast (轻提示)
操作成功确认，底部浮现，3 秒后消失
Info (信息)
需要关注的信息，顶部横幅
Warning (警告)
非阻断性警告
Error (错误)
需要用户操作的错误

Toast 规范

属性值
位置底部居中，距底部 24px
背景neutral-800
文字白色
圆角8px
内边距12px 16px
阴影0 4px 12px rgba(0,0,0,0.15)
动画淡入/淡出 200ms
持续时间3000ms

D12: 无障碍访问

决策：采用 WCAG 2.1 AA 级别标准

对比度要求

正文文字 ≥ 4.5:1
大号文字 (18px+) ≥ 3:1
非文字元素 ≥ 3:1
焦点指示器 ≥ 3:1

键盘导航

所有交互元素可 Tab 到达
可见的焦点指示器 (2px outline)
逻辑的 Tab 顺序
快捷键提示

屏幕阅读器

图标按钮有 aria-label
分组框有 label 关联
动态内容有 live region
表格有 header 关联
